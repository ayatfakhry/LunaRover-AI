# LunaRover-AI

LunaRover AI is a research-grade system designed for autonomous lunar rover operations. It enables a rover to navigate complex lunar terrain using AI and path planning algorithms, detect obstacles, calculate traversal costs, and monitor mission metrics via a real-time dashboard. The system ensures minimal human intervention while maintaining safety and efficiency.

Features
3D Terrain Mapping: Analyze and model lunar surface for navigation.
A Path Planning:* Calculate optimal paths avoiding obstacles.
Obstacle Detection & Traversal Cost: Identify hazards and compute difficulty of traversal.
Mission Dashboard: Monitor rover energy, battery, waypoints, and overall mission status.
Autonomous Execution: Rover follows waypoints and adapts in real-time to terrain changes.
Installation
git clone <repo_url>
cd LunaRover-AI
pip install -r requirements.txt
python main.py
Usage
Load 3D lunar terrain maps.
Execute autonomous path planning.
Monitor rover metrics in the mission dashboard.
Observe obstacle detection and traversal updates in real-time.
Technologies
Programming: Python 3.10+
Libraries: NumPy, Matplotlib, scikit-learn
Algorithms: A* path planning, AI/ML for terrain analysis
Visualization: 3D terrain and mission dashboard
